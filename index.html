Connect your Cloudflare Worker

The scorecard reads pre-calculated scores from your Cloudflare Worker. Follow these steps to get everything connected.

  1. 1
    Deploy the Worker — upload worker.js to a new Cloudflare Worker (same process as your proxy worker).
  2. 2
    Create a KV namespace — in your Worker settings, go to Settings → KV Namespace Bindings, create a namespace called SCORECARD_KV and bind it with variable name SCORECARD_KV.
  3. 3
    Add environment variables — in Settings → Variables, add: HUBSPOT_TOKEN (your token), FLEET_PROP (e.g. vehicle_count__contact_level_), ALLOWED_ORIGIN (your GitHub Pages URL), and optionally SYNC_SECRET (any password to protect manual syncs).
  4. 4
    Add a cron trigger — in Settings → Triggers → Cron Triggers, add 0 11 * * * to sync every morning at 7am ET.
  5. 5
    Enter your Worker URL below and click Save. Then click Sync now to run the first fetch.
Worker URL
Your Cloudflare Worker URL — found on the Worker's overview page. Do not include a trailing slash.
Sync secret (optional)
If you set a SYNC_SECRET environment variable on your Worker, enter the same value here to authorize manual syncs.

Scoring rules

To change scoring rules, edit the SCORING object at the top of worker.js and redeploy.
